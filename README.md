## 📌 Tentang NeoSINT Tracker

**NeoSINT Tracker** adalah platform OSINT (Open Source Intelligence) berbasis web yang dirancang khusus untuk pencarian dan verifikasi data publik di Indonesia. Dibangun untuk memudahkan akses informasi dari berbagai sumber terbuka.

**Website**: [neosint.online](https://neosint.online)

---

## 🔥 Fitur Lengkap

### 🎓 Data Pendidikan & Akademik
- **Cek Mahasiswa** - Verifikasi status mahasiswa aktif
- **Cek Dosen** - Cari data dosen berdasarkan NIDN
- **Cek Siswa** - Data siswa berdasarkan NISN
- **Cek Guru** - Verifikasi data tenaga pendidik
- **Cek Ijazah** - Validasi nomor ijazah

### 🆔 Data Kependudukan
- **Cek NIK** - Pencarian data KTP berdasarkan NIK
- **Cek Kartu Keluarga** - Verifikasi data KK
- **Cek NIK to Foto** - Cari foto berdasarkan NIK
- **Cek Data Keluarga** - Informasi anggota keluarga

### 💰 Data Keuangan & BPJS
- **Cek BPJSTK** - Verifikasi kepesertaan BPJS Ketenagakerjaan
- **Cek BPJS** - Cek status BPJS Kesehatan
- **Cek Rekening** - Validasi nomor rekening bank
- **Cek E-Wallet** - Cek kepemilikan e-wallet (OVO, GoPay, DANA, ShopeePay)

### 👤 Data Profil & Pekerjaan
- **Cek Pekerjaan** - Informasi riwayat pekerjaan
- **Cek Posisi** - Lacak posisi/alamat terakhir

### 🌐 Data Digital & Online
- **Cek Username** - Cari username di berbagai platform
- **Cek Domain** - Informasi kepemilikan domain
- **Cek Subdomain** - Discovery subdomain
- **Cek No Aktif** - Verifikasi nomor aktif
- **Cek Header** - Analisis header HTTP
- **Cek SS Web** - Screenshot website

### 📦 Tracking & Lainnya
- **Cek Resi** - Lacak paket pengiriman
